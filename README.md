# Monotonicity Constraints in Neural Networks

This repository provides a comprehensive framework for studying and benchmarking **monotonicity constraints in neural networks** using Python, NumPy, and PyTorch. These constraints are essential in applications where model predictions should preserve known monotonic relationships with respect to selected input features, especially in domains such as finance, healthcare, and safety-critical decision systems.
